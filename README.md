# dryboss.github.io

This is a personal portfolio website built with Next.js. It showcases projects, skills, and experience, and is designed for speed, accessibility, and easy deployment.

## Features

- Built with Next.js for performance and scalability
- Project and experience showcase sections
- Uses Geist font for a modern look
- Responsive and mobile-friendly design
- Hot reloading for efficient development
- Easy deployment to Vercel or any static hosting

## Getting Started

### Prerequisites

- Node.js (version 18 or newer)
- npm, yarn, pnpm, or bun

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/dryboss/dryboss.github.io.git
cd dryboss.github.io
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### Running the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Visit http://localhost:3000 to view your site.

## Project Structure

- app/ - Main application code (pages, components, styles)
- public/ - Static assets
- README.md - Project documentation

## Deployment

You can deploy this site on Vercel or any platform that supports Next.js. For more details, see the Next.js deployment documentation.
